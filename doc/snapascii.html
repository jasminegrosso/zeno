snapascii                         Convert binary N-body file to ascii

  in=???                          Input file name

  out=???                         Output file name

  options=Mass,Position,Velocity
				  Others are Potential,Aux,Key

  iformat=  %d                    Output format for integers

  rformat= %21.14E                Output format for floating-point

  VERSION=2.3                     Josh Barnes  11 June 1998
